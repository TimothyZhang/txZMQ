txZMQ - ØMQ for Twisted
=======================

txZMQ allows to integrate easily `ØMQ <http://zeromq.org>`_ sockets into
Twisted event loop (reactor).

txZMQ supports both CPython and PyPy, and ØMQ library version 2.2.x or 3.2.x.
